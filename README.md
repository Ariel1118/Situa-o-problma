    <h1>Situação-Problema Matemática</h1>
    <div class="box">
        <h2>Problema</h2>
        <p>
            Um treinador de basquete comprou novas bolas para o time.  
            Cada caixa contém <strong>6 bolas</strong>.  
            Ele comprou <strong>8 caixas</strong>.  
            Quantas bolas ao todo o treinador recebeu?
        </p>
    </div>
    <div class="box">
        <h2>Resolução</h2>
        <p>
            Para descobrir o total de bolas, basta multiplicar a quantidade de caixas pelo número de bolas em cada caixa:
        </p>
        <p>
            <strong>6 bolas × 8 caixas = 48 bolas</strong>
        </p>
        <div class="resultado">
            ✔️ <strong>O treinador recebeu 48 bolas de basquete.</strong>
        </div>
